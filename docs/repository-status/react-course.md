# react-course

## Estado actual

**Estado estimado:** material mixto de curso, apuntes y ejercicios.

No es un proyecto único. Es una carpeta madre con ejercicios de JS/React, apuntes y subproyectos separados, incluidos ejemplos de React Native con Expo.

## Evidencia encontrada

- raíz con archivos `00.js` a `04.js`
- `README.md`
  - describe apuntes de curso y temas por módulo
- `rn.md`
  - apuntes extensos de React Native
- carpeta `rn/`
  - ejemplos `rn01.js` a `rn07.js`
- subproyecto `project0/`
  - ejercicio TODO app con HTML/CSS/JS
- subproyecto `project1/`
  - app Expo / React Native tipo Pomodoro
- subproyecto `Contacts/`
  - app Expo / React Native de contactos

## Estado por subcarpeta relevante

### `project0/`

- ejercicio guiado de TODO app
- tiene `index.html`, `script.js`, `script2.js`, `styles.css`
- más orientado a práctica de fundamentos que a app final

### `project1/`

- proyecto Expo con `App.js`, `Timer.js`, `utils/`
- README con requerimientos claros de Pomodoro
- parece uno de los ejercicios más completos de la carpeta

### `Contacts/`

- proyecto Expo con varias pantallas y componentes
- incluye `App.js`, `ContactList.js`, `Row.js`, `screens/`
- parece otro ejercicio relativamente avanzado

## Lectura arquitectónica

- La carpeta funciona como **repositorio pedagógico**.
- Mezcla apuntes, mini ejercicios y apps chicas.
- Tiene bastante valor documental incluso si no está “cerrada” como producto.
